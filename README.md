# Nova Specs

Public specifications, invariants, schemas, and roadmap for the Nova blockchain.

> Status: Work in progress (research/design phase).

## Contents

- `specs/` — protocol specifications and invariants
- `schemas/` — data formats and compatibility notes
- `roadmap/` — milestones and phases
- `docs/` — explanations, diagrams, glossary

## Scope

This repository focuses on **design documentation**.  
The core Rust implementation is tracked separately (private for now).
